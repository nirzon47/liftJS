var has functional scope



JavaScript only hoists the declarations of functions and variables into memory during compile time, not the assignments.

This means that if a variable is declared and initialized after using it, the value will be undefined. 

function doSomething() {
 var doing = "something";
}
console.log(doing);

What should console.log print?

  • something
  • Reference Error
Correct!

The variable can only be accessed within the function.

Incorrect!

The variable exists only within the function. It can not be accessed from outside.
